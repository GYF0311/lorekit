# _工作台

过程文件的家。**这里的东西不会进向量索引**，也不会被 Agent 当作"事实"来引用。你可以随便乱扔。

## 四个子目录

| 目录        | 用途                                              | 过期策略 |
|-------------|---------------------------------------------------|----------|
| `00_收件/`  | Inbox。想法、摘录、随手贴，没整理过的东西         | 7 天     |
| `10_草稿/`  | 正在写的文章/笔记/计划，迟早会搬到 80_写作/ 去    | 30 天    |
| `20_临时/`  | 临时工作文件、中间产物、拼图碎片                  | 14 天    |
| `30_待整理/`| 明确要归档但还没决定去哪的条目                    | 3 天     |

## 过期策略怎么跑

`wiki doctor` / lint 扫到过期文件时会：

1. **00_收件 > 7 天**：提示你"这条要不要归档？要就决定主语，不要就移到 `_archive/`"
2. **10_草稿 > 30 天**：提示你"这个草稿还写吗？写就升级到 `80_写作/`，不写就 archive"
3. **20_临时 > 14 天**：默认直接移到 `_archive/`
4. **30_待整理 > 3 天**：强制你做决定，不允许悬停

## 规则

- **不要在这里直接引用双链**——其它目录的页面不应该 link 到工作台
- **不要放原始资料**——原文放 `60_来源/`
- **不要当长期记忆用**——这里是过道，不是房间
