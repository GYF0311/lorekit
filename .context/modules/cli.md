# 模块：CLI

## 职责
注册 `lorekit` 命令面，读取版本信息，展示无参数 banner，并统一 commander 的退出码语义。

## 负责路径
- `src/cli.ts`
- `bin/lorekit.js`
- `VERSION`
- `package.json`
- `dist/` build output, regenerated by `npm run build`

## 关键契约
- 项目是 ESM TypeScript，运行时 Node.js >= 18。
- 无参数 `lorekit` 显示 banner 和 best-effort corpus/vector 状态。
- commander 参数/用法错误退出 `2`；运行时错误退出 `1`。
- 源码里人类输出统一走 `src/utils/logger.ts`，不要新增直接 `console.log`。
- `dist/` 不手工改；改源码后跑 `npm run build`。

## 关联模块
- 调用 `corpus-core`、`fetch-ingest`、`sync-search-vector`、`safety-maintenance`、`skills-agent`、`obsidian-gbrain`。
- 命令行为变化通常还要看 `docs-tests-release`。

## 读什么
- `docs/CONVENTIONS.md`：stdout/stderr、logger、exit code 规则。
- `docs/CODEBASE-MAP.md`：命令归属。
- 具体命令文件：`src/commands/<name>.ts`。

## 验证
- `npm run verify`
- `node dist/cli.js --help`：构建后确认命令注册。
