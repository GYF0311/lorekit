# _工作台

过程文件的家。**这里的东西不会进向量索引**，也不会被 Agent 当作"事实"来引用。

## 四个子目录

| 目录 | 用途 | 过期策略 |
|---|---|---|
| `收件/` | Inbox。想法、摘录、随手贴 | 7 天 |
| `草稿/` | 正在写的文章/笔记/计划 | 30 天 |
| `临时/` | 临时工作文件、中间产物 | 14 天 |
| `待整理/` | 明确要归档但还没决定去哪 | 3 天 |

## 过期策略

`lorekit doctor` 扫到过期文件时会提醒：

1. **收件 > 7 天**：提示"要归档还是 archive？"
2. **草稿 > 30 天**：提示"还写吗？升级到 `写作/` 还是 archive？"
3. **临时 > 14 天**：建议移到 `_归档/`
4. **待整理 > 3 天**：强制做决定，不允许悬停

## 规则

- 不要在这里建双链——其它目录不应该 link 到工作台
- 不要放原始资料——原文放 `原料/`
- 不要当长期记忆用——这里是过道，不是房间
